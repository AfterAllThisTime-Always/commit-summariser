# Commit Summariser

A VS Code extension that integrates with **Git Graph** to provide commit summaries.

### Features
- Right-click any commit in Git Graph → **Summarize Commit**
- Automatically calls GitHub Copilot CLI (`gh copilot explain`) to summarise commit diffs
- Displays output in a markdown tab

### Usage
1. Install **Git Graph** extension
2. Right-click a commit → `Summarize Commit`
3. Read the AI-generated summary

### Requirements
- Git installed
- GitHub CLI (`gh`) with Copilot access
